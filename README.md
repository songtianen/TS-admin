# ts-admin

## Description

RBAC 基于角色权限控制管理系统 https://www.card12.com

## QuickStart

#### Development

```javaScript
 npm i
 npm install
 npm run dev-server
 npm run dev-client
```

#### Production

```javaScript
 npm run build-server
 npm run build-client
```

#### Deploy

```javaScript
npm run build-client
pm2 deploy ecosystem.json production setup
pm2 deploy ecosystem.json production
```
